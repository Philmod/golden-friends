# Game Rules

Golden Friends is based on the French TV show "Une Famille en Or" (Family Feud).

## Teams

- **Girls Team** (Pink)
- **Boys Team** (Blue)

Each team can have multiple players who share a single buzzer interface.

## Game Phases

### 1. Lobby (`lobby`)

Players join the game by:
1. Scanning the QR code on the landing page
2. Entering their name
3. Selecting their team (Girls or Boys)

The admin waits until all players have joined before starting.

### 2. Face-Off (`faceoff`)

Two players (one from each team) compete to buzz first:

1. Admin shows the question (hidden until ready)
2. Admin unlocks buzzers
3. Players race to buzz
4. First buzzer wins control for their team
5. If the first answer is wrong, the other team's buzzer can answer

### 3. Play (`play`)

The controlling team tries to find all answers:

1. Team members take turns guessing
2. Correct answers are revealed on the board
3. Wrong answers add a strike (X)
4. Points accumulate in the round total

### 4. Steal (`steal`)

Triggered when the controlling team gets 3 strikes:

1. The opposing team gets ONE chance to steal
2. They must guess any remaining answer
3. If correct: they win ALL round points
4. If wrong: original team keeps the points

### 5. Reveal (`reveal`)

After a round ends:

1. All remaining answers are revealed
2. Points are awarded to the winning team
3. Admin can proceed to the next question

### 6. Complete (`complete`)

Game is finished. Final scores displayed.

## Scoring

### Multiple Choice Questions

Each answer has a point value (typically 5-50 points). Higher-ranked answers are worth more.

| Rank | Typical Points |
|------|----------------|
| #1 | 35-50 |
| #2 | 25-30 |
| #3 | 15-20 |
| #4 | 10-15 |
| #5 | 5-10 |

### Buzzer Questions

Quick-fire questions with a single correct answer:

- **Correct**: +30 points
- **Wrong**: -10 points (next player can try)

### Point Multipliers

Questions can have multipliers for higher stakes:

| Round Type | Multiplier |
|------------|------------|
| Normal | 1x |
| Double | 2x |
| Triple | 3x |

## Strikes

- 3 strikes = lose control, other team can steal
- Strikes reset at the start of each question
- Visual: Big red X on TV display
- Sound: Strike buzzer

## Timer

For buzzer questions, the admin can start a countdown timer:

- Players must buzz before time runs out
- Timer sound plays when time expires
- Timer can be stopped manually

## Drinking Rules (Optional)

When enabled, special rules display on TV:

- **Strike**: Player who caused it drinks
- **#1 Answer**: Everyone else drinks
- **Steal Success**: Losing team drinks
- **Steal Fail**: Stealing team drinks

## Tips for Playing

### For Players
- Join early to test your buzzer works
- Wait for the question to be fully revealed before buzzing
- On mobile, hold the phone firmly for haptic feedback

### For Admins
- Have questions ready before starting
- Use the lock/unlock buttons to control buzzer access
- Award points before moving to next question
- Use the drinking rules toggle for party atmosphere
